# ❤️ Cardiovascular Disease Prediction

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Healthcare](https://img.shields.io/badge/Healthcare-Cardiology-red.svg)

## 📖 Overview

Binary classification model to predict cardiovascular disease risk using patient health metrics including blood pressure, cholesterol, glucose levels, lifestyle factors, and medical history.

## 🎯 Features

- CVD risk prediction
- Risk factor analysis
- Early detection support
- Preventive care insights

## 🛠️ Tech Stack

- Scikit-learn, XGBoost
- Pandas, NumPy
- Matplotlib, Seaborn

## 🚀 Usage

```bash
git clone https://github.com/adityavdn/Cardiovascular-Disease-Prediction.git
pip install -r requirements.txt
jupyter notebook
```

## 📊 Health Metrics

- Blood Pressure, Cholesterol
- Glucose, BMI
- Age, Smoking Status
- Physical Activity
